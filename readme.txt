== Description ==

This plugin adds to your site a payment method via the PlatBox Online Payment System and implements the PlatBox API in your store..

Cross-platform billing and payment business solution for project monetization in one single integration

Most popular payment methods for business with high CTR

PayBox White label
Customizable platform configuration and design to match project overall style

== Installation ==

1. Upload Archive contents to the `/wp-content/plugins/platbox/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adding PlatBox payment method to WooCommerce settings.
4. Setting PlatBox payment method
